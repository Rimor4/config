# 配置方法

## Windows

### Powershell

将`Microsoft.PowerShell_profile.ps1`复制到在pwsh中运行`$PROFILE`得到的路径

### Vim

1. `.vimrc` `.ideavimrc`复制到用户目录下
2. 安装vim插件管理器后运行对应命令进行插件安装

### AutoHotKey

将`*.ahk`文件添加到`shell:startup`路径（自启动）。

## Linux

### Vim

1. `.vimrc` 复制到`~`目录下
2. 安装vim插件管理器后运行对应命令进行插件安装

### Zsh

`.zshrc`复制到`~`目录下
